遇到的难点

📄

页面显示空白

HTML 结构或 CSS 样式配置错误，导致页面内容无法正常渲染显示

🌍

中英文切换不彻底

部分动态内容或 JavaScript 生成的文本遗漏翻译键，导致切换语言时显示不完整

🔤

网页出现乱码

字符编码设置不正确，特殊字符和中文内容显示为乱码或方块

📌

二级菜单无法展开或跳转

下拉菜单事件处理或页面路由逻辑错误，导致导航功能失效

💻

网页只能本地查看

文件使用本地路径引用，未部署到服务器，其他设备无法通过链接访问

🖼️

本机图片上传无法在外部查看

使用 FileReader 读取本地图片，仅在浏览器会话中有效，刷新后数据丢失
