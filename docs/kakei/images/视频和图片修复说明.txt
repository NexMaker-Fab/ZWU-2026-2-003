## ⚠️ 重要提示

### 关于 work10.mov 格式

**问题：** `.mov` 格式在某些浏览器中可能无法直接播放。

**建议解决方案：**
1. **转换为 .mp4 格式**（推荐）
   - 使用在线工具：https://cloudconvert.com/mov-to-mp4
   - 或使用软件：HandBrake、FFmpeg
   
2. **转换后重命名**
   ```
   work10.mov → work10.mp4
   ```

3. **更新代码**
   如果转换了格式，需要修改 works.html 中的配置：
   ```javascript
   'work10': {
       title: '创意技术与AI',
       media: [
           { type: 'video', src: 'images/work10.mp4' }  // 改为 .mp4
       ]
   },
   ```

## 🎬 视频播放说明

### 支持的视频格式
- ✅ **.mp4** (H.264编码) - 最佳兼容性
- ⚠️ **.mov** - 部分浏览器可能不支持

### 视频规格建议
- **分辨率**: 1920x1080 (1080p) 或 1280x720 (720p)
- **文件大小**: < 10MB
- **时长**: 10-60秒
- **编码**: H.264 + AAC音频

## ❓ 常见问题

### Q1: 视频不播放怎么办？
**A:** 
1. 确认文件格式是 .mp4
2. 如果是 .mov，建议转换为 .mp4
3. 检查浏览器控制台是否有错误

### Q2: 图片显示不正确？
**A:**
1. 确认文件名正确（workX-N.jpg）
2. 确认文件在 images 文件夹中
3. 刷新浏览器（Ctrl+F5）

### Q3: 想为某个作品添加更多图片？
**A:**
1. 将新图片放入 images 文件夹
2. 按照上面的"如何添加更多图片"步骤更新代码
