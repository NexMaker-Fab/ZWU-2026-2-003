Final Project

ASD 多感官互动沙盘系统

本项目面向 ASD 儿童，融合触觉、视觉、听觉与动态反馈，打造一个低刺激、可探索、具有安全感的沉浸式互动沙盘系统。

项目概念 成品视频 海报展板

CONCEPT

项目概念与核心价值

一、项目背景与痛点

自闭症儿童存在社交、情境理解与规则学习三重障碍，真实公共场景（如公交站台）易引发焦虑与失控。现有卡片、ABA、APP 与简易教具多为场景抽象、依赖人工、反馈单一，训练难以迁移到现实行为。本项目以公交站台为载体，打造低压力实体交互康复装置，契合 SDG3 健康、SDG4 特教、SDG10 平等发展目标。

二、三维设计创新

  1. 场景创新：实体还原进站 → 排队 → 乘车完整流程，采用柔和云朵视觉降低感官刺激，强化现实行为迁移。
  2. 交互创新：基于 Arduino 与多类型传感器构建多模态闭环系统，灯光、语音与触觉同步反馈，动作即可自主触发引导。
  3. 方法创新：使用状态机拆分分段训练任务并游戏化情境设计，提高儿童配合度与自主规则学习能力。

三、材料与技术实现

系统采用“木质底座 + 毛毡场景 + 电子交互”三层结构：

  1. 主材：木板承重、羊毛毡弱化刺激、粘土丰富场景细节。
  2. 硬件：刷卡、压力传感器与按钮采集行为；LED 灯带、语音与画面系统联动输出反馈。
  3. 程序：三段式状态机控制训练流程，并针对信号延迟与误触发进行调试优化。

四、市场与社会价值

市面缺少完整生活化的实体模拟教具，本装置以低成本填补市场空白。可控模拟环境减少外出焦虑，将抽象公共规则转化为可操作任务；降低特教机构康复设备门槛，推动包容性特殊教育，为自闭症社交辅助训练提供可落地的实践方案。

PROJECT SUMMARY

云朵车站项目简介

情境式训练场景

云朵车站是面向自闭症谱系障碍儿童研发的情境式交互康复训练系统，融合特殊教育与交互设计，以日常生活中的公交车站为原型搭建实体模拟场景，围绕进站、排队、乘车三大出行行为开展训练。

低压力交互体验

项目采用柔和云朵主题视觉风格，打造低压力、高安全感的训练环境，通过灯光、语音、物理传感交互，引导儿童自主学习公共场合行为规则，改善社交适应能力。

创新实现与价值

本项目通过实体公交实景训练空间、Arduino 多模态闭环交互系统和结构化任务拆解，将复杂社交行为分解为可执行训练单元，结合友好的视觉设计降低儿童抵触心理，提升生活技能迁移与参与度。

装置构建与可持续价值

装置以木质板材为基础框架，羊毛毡、粘土构建柔和卡通场景，依靠传感器、刷卡模块采集行为数据，配合 LED 灯带、语音画面系统提供视听引导，经软硬件调试保障长期稳定运行。

项目依托情境干预与特殊教育辅助技术相关理论，契合 SDG3、SDG4、SDG10 目标，以低成本实体交互方案填补实景康复教具市场空白，为特教机构与家庭提供普惠型康复工具。
