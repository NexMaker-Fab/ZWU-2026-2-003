# Sophia 个人作品集 - 图片上传指南

## 📸 如何添加您的照片

### 方法一：直接替换（最简单）

#### 1. 添加头像照片

找到 HTML 文件中的这一行（大约在第 485 行）：
```html
<div class="profile-avatar">👩‍🎨</div>
```

**替换为：**
```html
<img src="your-photo.jpg" alt="Sophia" style="width: 200px; height: 200px; object-fit: cover; border-radius: 50%;">
```

**操作步骤：**
1. 准备一张您的照片（建议正方形，至少 400x400 像素）
2. 将照片命名为 `your-photo.jpg`
3. 把照片文件和 `sophia_portfolio.html` 放在**同一个文件夹**里
4. 用记事本打开 HTML 文件，找到上面的代码，替换即可

---

#### 2. 添加作品图片

找到作品卡片部分（大约在第 570-600 行），有三个作品卡片：

**第一个作品：**
```html
<div class="project-thumbnail">🎨</div>
```
**替换为：**
```html
<img src="project1.jpg" alt="品牌视觉设计" style="width: 100%; height: 100%; object-fit: cover;">
```

**第二个作品：**
```html
<div class="project-thumbnail" style="background: linear-gradient(135deg, #C9B1FF, #A8D8EA);">✨</div>
```
**替换为：**
```html
<img src="project2.jpg" alt="交互艺术装置" style="width: 100%; height: 100%; object-fit: cover;">
```

**第三个作品：**
```html
<div class="project-thumbnail" style="background: linear-gradient(135deg, #F4A6C9, #C9B1FF);">🎬</div>
```
**替换为：**
```html
<img src="project3.jpg" alt="动态影像作品" style="width: 100%; height: 100%; object-fit: cover;">
```

---

### 方法二：使用在线图片（无需下载）

如果您有图片链接（比如来自图床、云存储等），可以直接使用 URL：

```html
<img src="https://example.com/your-photo.jpg" alt="Sophia" style="width: 200px; height: 200px; object-fit: cover; border-radius: 50%;">
```

## 💡 图片建议尺寸

| 位置 | 建议尺寸 | 格式 |
|------|---------|------|
| 头像 | 400x400px（正方形） | JPG/PNG |
| 作品缩略图 | 800x600px（横向） | JPG/PNG |
| 爱好图片 | 400x400px（正方形） | JPG/PNG |

## 🔧 快速测试

修改后，只需：
1. 保存 HTML 文件
2. 双击打开 `sophia_portfolio.html`
3. 在浏览器中查看效果

如果图片没有显示，检查：
- ✓ 图片文件名是否正确
- ✓ 图片是否和 HTML 文件在同一文件夹
- ✓ 文件路径是否有拼写错误
