🔧 Arduino Uno R4 WiFi 开发板介绍

Arduino Uno R4 WiFi 是 Arduino 官方推出的最新一代 Uno 系列开发板，基于 Renesas RA4M1 微控制器，集成了 WiFi 和蓝牙功能，性能更强、功能更丰富。

主要特性

  • 处理器：Renesas RA4M1 (ARM Cortex-M4)，48MHz 主频
  • 内存：32KB SRAM + 256KB Flash
  • 无线连接：集成 ESP32-S3 模块，支持 WiFi 和 Bluetooth Low Energy (BLE)
  • I/O 引脚：14个数字I/O（其中6个PWM），6个模拟输入
  • 通信接口：UART、I2C、SPI、USB-C
  • 供电：通过 USB-C 或 VIN 引脚供电（7-24V）
  • 尺寸：与经典 Uno 兼容，可直接使用现有 Shield

📌 应用场景

Arduino Uno R4 WiFi 适用于物联网项目、智能家居控制、远程数据采集、无线传感器网络等需要网络连接的场景。相比传统 Uno，它无需额外的 WiFi 模块即可实现联网功能。

💻 Arduino IDE 基础操作介绍

在开始编写代码之前，首先需要了解 Arduino IDE 的基本界面和操作。Arduino IDE 是官方提供的集成开发环境，用于编写、编译和上传代码到 Arduino 开发板。

📌 第一步：认识 Arduino IDE 界面

打开 Arduino IDE 后，可以看到主界面包含以下几个重要区域：

  • 菜单栏：位于顶部，包含文件、编辑、项目、工具等选项
  • 工具栏：包含常用功能按钮，如验证（✓）、上传（→）、新建、打开、保存等
  • 代码编辑区：中间最大的区域，用于编写和编辑 Arduino 代码
  • 消息输出区：底部显示编译信息、错误提示和串口通信数据

图1：Arduino IDE 主界面及操作区域说明

📌 第二步：选择正确的开发板和端口

在上传代码之前，必须确保选择了正确的开发板型号和通信端口：

  1. 点击菜单栏的 工具（Tools）
  2. 选择 开发板（Board） → 找到并选择 Arduino Uno R4 WiFi（或其他你使用的型号）
  3. 选择 端口（Port） → 选择已连接的 COM 端口（Windows）或 /dev/ttyUSB*（Linux/Mac）

⚠️ 注意：如果找不到对应的端口，请检查 USB 线是否连接好，并确保安装了相应的驱动程序。

📌 第三步：理解 setup() 和 loop() 函数

每个 Arduino 程序都必须包含两个核心函数：setup() 和 loop()。它们是程序的入口点和运行主体。

setup() 函数

作用：只在程序开始时运行一次，用于初始化设置。

常见用途：设置引脚模式、初始化串口通信、配置传感器等。

示例：

  • pinMode(LED_BUILTIN, OUTPUT) - 设置LED引脚为输出模式
  • Serial.begin(9600) - 启动串口通信
loop() 函数

作用：在 setup() 执行完后，会无限循环运行，是程序的主要逻辑部分。

常见用途：读取传感器数据、控制输出设备、执行判断逻辑等。

示例：

  • digitalWrite(LED_BUILTIN, HIGH) - 点亮LED
  • delay(1000) - 延时1秒
  • analogRead(A0) - 读取模拟引脚数据
void setup() {
    // Runs only once here
}

void loop() {
    // Loops repeatedly here
}

图3：Arduino 程序基本结构 - setup() 和 loop() 函数示例

简单例子：如果你想让板载LED闪烁，可以在 setup() 中设置引脚模式，在 loop() 中反复开关LED并延时。

#include <FastLED.h>

// 灯环配置
#define NUM_LEDS 12
#define DATA_PIN 6
CRGB leds[NUM_LEDS];

// 超声波传感器引脚
#define TRIG_PIN 9
#define ECHO_PIN 10

// 声音传感器引脚
#define SOUND_SENSOR A0

// 模式变量
enum Mode { PROXIMITY, BLOWING };
Mode currentMode = PROXIMITY;

// 超声波相关变量
long duration;
int distance;
const int MAX_DISTANCE = 50; // 最大检测距离50cm
const int MIN_DISTANCE = 5;  // 最小检测距离5cm

// 颜色状态
int colorState = 0; // 0:蓝色, 1:红色, 2:绿色
bool colorChanged = false;

// 吹气模式变量
unsigned long lastRotationTime = 0;
const int ROTATION_DELAY = 200; // 旋转延迟(毫秒)
int rotationOffset = 0;

// 声音检测
const int SOUND_THRESHOLD = 500; // 声音阈值，需要根据实际情况调整
unsigned long lastBlowTime = 0;
const unsigned long BLOW_MODE_DURATION = 5000; // 吹气模式持续时间5秒

void setup() {
  // 初始化串口通信
  Serial.begin(9600);
  
  // 初始化超声波引脚
  pinMode(TRIG_PIN, OUTPUT);
  pinMode(ECHO_PIN, INPUT);
  
  // 初始化声音传感器引脚
  pinMode(SOUND_SENSOR, INPUT);
  
  // 初始化LED灯环
  FastLED.addLeds<WS2812, DATA_PIN, GRB>(leds, NUM_LEDS);
  FastLED.setBrightness(100);
  
  // 初始全部熄灭
  clearLEDs();
  FastLED.show();
}

void loop() {
  // 检测声音
  checkSound();
  
  // 模式切换逻辑
  if (currentMode == BLOWING && millis() - lastBlowTime > BLOW_MODE_DURATION) {
    currentMode = PROXIMITY;
    clearLEDs();
  }
  
  // 根据当前模式执行相应的灯光效果
  if (currentMode == PROXIMITY) {
    proximityMode();
  } else {
    blowingMode();
  }
  
  FastLED.show();
  delay(50);
}

void checkSound() {
  int soundValue = analogRead(SOUND_SENSOR);
  
  if (soundValue > SOUND_THRESHOLD) {
    currentMode = BLOWING;
    lastBlowTime = millis();
    rotationOffset = 0;
  }
}

void proximityMode() {
  // 读取超声波距离
  digitalWrite(TRIG_PIN, LOW);
  delayMicroseconds(2);
  digitalWrite(TRIG_PIN, HIGH);
  delayMicroseconds(10);
  digitalWrite(TRIG_PIN, LOW);
  
  duration = pulseIn(ECHO_PIN, HIGH);
  distance = duration * 0.034 / 2;
  
  // 限制距离范围
  distance = constrain(distance, MIN_DISTANCE, MAX_DISTANCE);
  
  // 计算要点亮的LED数量
  int ledsToLight = map(distance, MIN_DISTANCE, MAX_DISTANCE, NUM_LEDS, 1);
  ledsToLight = constrain(ledsToLight, 1, NUM_LEDS);
  
  // 当所有LED都点亮时，准备切换颜色
  if (ledsToLight == NUM_LEDS && !colorChanged) {
    colorState = (colorState + 1) % 3;
    colorChanged = true;
  }
  
  // 当LED数量减少时，重置颜色切换标志
  if (ledsToLight < NUM_LEDS) {
    colorChanged = false;
  }
  
  // 根据颜色状态设置颜色
  CRGB currentColor;
  switch (colorState) {
    case 0: currentColor = CRGB::Blue; break;
    case 1: currentColor = CRGB::Red; break;
    case 2: currentColor = CRGB::Green; break;
  }
  
  // 清除所有LED
  clearLEDs();
  
  // 点亮相应数量的LED
  for (int i = 0; i < ledsToLight; i++) {
    leds[i] = currentColor;
  }
  
  // 串口输出调试信息
  Serial.print("Distance: ");
  Serial.print(distance);
  Serial.print("cm, LEDs: ");
  Serial.print(ledsToLight);
  Serial.print(", Color: ");
  Serial.println(colorState);
}

void blowingMode() {
  if (millis() - lastRotationTime > ROTATION_DELAY) {
    // 清除所有LED
    clearLEDs();
    
    // 设置三个色块，每个色块4个LED
    for (int i = 0; i < NUM_LEDS; i++) {
      int pos = (i + rotationOffset) % NUM_LEDS;
      
      if (pos < 4) {
        leds[i] = CRGB::Blue;    // 蓝色块
      } else if (pos < 8) {
        leds[i] = CRGB::Red;     // 红色块
      } else {
        leds[i] = CRGB::Green;   // 绿色块
      }
    }
    
    // 更新旋转偏移量
    rotationOffset = (rotationOffset + 1) % NUM_LEDS;
    lastRotationTime = millis();
  }
}

void clearLEDs() {
  for (int i = 0; i < NUM_LEDS; i++) {
    leds[i] = CRGB::Black;
  }
}

🔧 运用到的 Arduino 硬件分析

1. HC-SR04超声波传感器

HC-SR04是一款常用的超声波测距模块，通过发射40kHz超声波并接收回波来测量物体距离。其工作原理是：Trig引脚发送一个10μs的高电平触发信号，模块自动发射8个40kHz的方波，Echo引脚输出高电平，高电平的持续时间即为超声波往返时间。通过公式“距离 = (高电平时间 × 声速) / 2”可计算出距离（声速约为340m/s）。HC-SR04的测量范围为2cm至400cm，精度可达3mm，广泛应用于避障机器人、智能小车、液位检测及距离感应等场景。

2. WS2812灯环

WS2812灯环是将多颗可寻址LED集成于一体的环形RGB灯板，每颗灯珠内置驱动芯片，仅需一根信号线即可独立控制每颗灯珠的任意颜色和亮度。使用时将VCC接5V、GND接地、DI接Arduino的数字引脚，配合FastLED库即可轻松实现跑马灯、呼吸灯等炫彩效果。常用于指示灯、氛围灯、创意装饰及互动灯光项目中。

3. HS-S05B声音传感器

HS-S05B声音传感器是一种高灵敏度的声音检测模块，能够识别环境中的声音信号，如拍手声、吹气声等。使用时将VCC接5V、GND接地、A接Arduino的模拟输入引脚，通过analogRead()函数读取声音强度值，可根据阈值判断是否检测到声音。常用于声控开关、互动装置、音乐节奏检测及语音控制等场景。

应用场景展示

通过HC-SR04超声波传感器测量距离，根据距离远近动态控制WS2812灯环点亮的LED数量和颜色；同时通过HS-S05B声音传感器检测吹气声音，触发灯环旋转模式。装置可用于创意装饰、智能感应灯、互动展示等场景，实现距离感应和声控互动的双重功能。
